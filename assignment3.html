Task 3

What was the issue?

Even though the variable "secret" in the code looks completely random, it's really hard to turn it into 5

How did you solve it?

The bug in this code is that the value of 'secret' is only randomly generated once at the beginning of the program and does not change after that. So if A 5 is not generated the first time, no matter how many times button A is pressed, a 5 will not appear. You only need to change the position of the random number set from the boot time to the press of button A.
